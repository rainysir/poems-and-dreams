# Poems-and-dreams

## 简介
基于react+ts，包含博客等功能

> * 配制cnpm（[更多--私有NPM访问](http://npm.iszhy.com)）
		```
		cnpm config set registry http://npm.iszhy.com
    ```

> * 安装yam和其他依赖
    ```
    yarn or npm i
    ```

> * 本地开发启动
    ```
    npm start
    ```

> * 发布上线
    ```
    npm run build && npm run publish ${password}
    ```
